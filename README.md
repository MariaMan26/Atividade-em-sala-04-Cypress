# Atividade-em-sala-04-Cypress
Esse repositório foi criado com o intuito de enviar a atividade realizada na disciplina teste de front-end no que diz respeito a teste E2E, fora solicitado escrever um código HTML para o teste.js enviado pelo professor.

Além de desenvolver minhas práticas com html e testes também compreendi melhor o conceito de TDD.

Para rodar o arquivo teste.js no Cypress (ferramenta que está sendo utilizada pela turma), ele precisaria estar dentro da pasta do Cypress. Essa pasta não foi enviada junto com as demais dependências para não deixar o arquivo ZIP mais pesado.
